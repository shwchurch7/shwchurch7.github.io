北京守望教会主日报告事项（2019年3月3日）

北京守望教会主日报告事项（2019年3月3日）

 

一.通知

1.主日聚会时间、地点：

A.8：30在中关村广场（三层平台）敬拜；

B.教会特别提醒，去平台敬拜的弟兄姊妹，要在主面前充分祷告，凭着信心面对各样的挑战，同时做好相应的准备。愿主与我们同在！

C.户外敬拜已近8年，我们依旧需要持守那从天而来的异象。为此，我们恳求主，坚固我们的信心，挑旺我们的盼望，复兴我们的祷告，使我们在新的争战阶段同样能站立得稳。相信主必为我们开出路，使我们能早日进入新堂，恢复全教会一同聚集在祂面前的敬拜；

2.奉献：

A.户外敬拜期间，十一奉献可以通过现场或转账的方式。有需要通过转账奉献的弟兄姊妹，请联系四萍姊妹（18519224079）；

B.现场奉献的弟兄姊妹，可以选择主日、祷告会或圣餐聚会时在长远奉献；

C.有给个人的指名奉献或给某项事工的专项奉献，请放在专用的信封里并写明奉献用途，以便与十一奉献区分开；

3.圣餐：3月圣餐礼拜5、6日（礼拜二、三）晚19:00在长远会堂举行，请弟兄姊妹安排好时间参加；

二.通报

1.户外敬拜及教会动态：

A.2月24日，是自4.10以来的第412次户外敬拜。本主日没有弟兄姊妹到平台敬拜；

B.阿迟姊妹自3月起停止教会服侍，其工作由岩松弟兄接替；

2.财务报告：19年1月教会财务报告如下：

A.总收入315,735.85元。其中奉献收入276,718.71元（包含十一奉献收入276,618.71元，专项奉献收入100.00元），其他收入 39,017.14元；

B.总支出208,077.20元。其中包含房租支出71,657.14元，差旅费 13,749.50元；

3.下主日证道：

A.经文：列王纪上19章；

B.讲员：曾淼传道；

守望教会办公室
